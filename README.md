## Installation

Install dependencies 
```terminal
npm i
``` 
Open up MySQL shell and input 
```terminal
source db/schema.sql
```

Then quit MySQL shell and input the following in your terminal
```terminal
npm run seed
```
To start running application simply input 
```terminal
node server.js
```
